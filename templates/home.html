Currently Working On

My focus lately has been getting more experience with machine learning and mastering techniques for processing large amounts of data, using programs such as using MapReduce and Spark. I have also been learning how to utilize algorithms such as Frequent Itemsets and Association Rules, Near Neighbor Search in High Dimensional Data, Locality Sensitive Hashing (LSH), Dimensionality Reduction, and Recommendation Systems.
One of my current academic projects includes building a Content and Collaborative Based Deep Learning Recommendation System with my colleagues. Our recommendation system will consist of multiple neural networks, as well as a linear regression and K-Means model. We will be utilizing the SmartcastTV proprietary dataset, as Smartcast TV is interested in using our research to improve their current recommendation system. Our goal is to create a recommendation system that can tackle efficiently tackle the Long Tail and Cold Start problems common to recommendation systems efficiently.
