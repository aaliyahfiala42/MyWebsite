Currently Working On

My focus lately has been getting more experience with machine learning and mastering techniques for processing large amounts of data, such as using MapReduce, Spark. As well as learning how to utilize algorithms such as Frequent itemsets and Association rules, Near Neighbor Search in High Dimensional Data, Locality Sensitive Hashing (LSH), Dimensionality reduction, and Recommendation Systems.
One of my current academic projects includes building a Content and Collaborative Based Deep Learning Recommendation System with my colleagues. Our recommendation system will consist of multiple neural networks, as well as a linear regression and K-Means model. We will be utilizing the SmartcastTV proprietary dataset , as they are interested in using our research to develop a recommendation system that is able to tackle the Long Tail and Cold Start problems common to recommendation systems efficiently.
